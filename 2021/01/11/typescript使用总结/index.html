Type Script使用总结

  • 安装typescript
sudo apt install npm
sudo npm install -g typescript

通过tsc命令确认安装完成

tsc -v
  • 运行ts文件
tsc ***.ts // 得到***.js文件
node ***.js // 使用node命令执行js文件

  • 运行ts文件出现以下错误

    Cannot find module 'fs'

    执行以下命令：

    sudo npm install @types/node
